# ascii-portscan

Lightweight ASCII curses port scanner.

## Features

- Scans TCP ports
- Shows progress in a curses UI
- Lists open ports
- Pure Python stdlib
- No external dependencies

## Usage

```bash
python3 main.py
python3 main.py 127.0.0.1
python3 main.py example.com
```

Press `q` to stop while scanning.

## Notes

Use only on systems you own or have permission to test.
